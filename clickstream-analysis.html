Customer Insights via Clickstream Analysis

Problem

Product and marketing teams needed a clear understanding of how users navigate the platform and where drop-offs were occurring.

Approach

  • Ingested clickstream event data stored in AWS S3
  • Sessionized user activity using Python and SQL
  • Built funnel and path analysis metrics

Tools Used

  • AWS S3
  • Python
  • SQL
  • Power BI

Impact

Identified high-friction user journeys and enabled data-driven UX improvements that improved engagement.
